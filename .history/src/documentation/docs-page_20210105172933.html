RamdanRiawan/DB adalah class yang oke gan untuk mengelola data dari database, mantull!!!

Inisialisasi Class:

Sebelum memulai project kalian wajib menginisialisasi classnya boy

<?php

include "./vendor/autoload.php";

use RamdanRiawan\DB;

$db = new DB('localhost', 'root', '', 'database');

1.1 Mengambil Semua Data Dari Tabel

Setelah menginisialisasi class maka semua tabel akan otomatis terdeteksi. Disini kita menggunakan sample table data_admin

<?php

$db->data_admin();
Result: 

Array
(
    [0] => stdClass Object
        (
            [id_admin] => 1
            [nama_admin] => admin
            [username] => admin
            [password] => 21232f297a57a5a743894a0e4a801fc3
        )

    [1] => stdClass Object
        (
            [id_admin] => ADM20210104051006553
            [nama_admin] => putra
            [username] => putra
            [password] => 5e0c5a0bf82decdd43b2150b622c66c5
        )

)

1.2 Mengambil satu data dari tabel berdasarkan Primary Key

Default primary key adalah berdasarkan nama tabel untuk data_ diganti jadi id_. misal data_admin maka primary key harus id_admin. Berikut adalah kode untuk menampilkan data admin berdasarkan id yg dicari.

<?php

$db->data_admin('1');
Result: 

stdClass Object
(
	[id_admin] => 1
	[nama_admin] => admin
	[username] => admin
	[password] => 21232f297a57a5a743894a0e4a801fc3
)

1.3 Mengambil satu data berdasarkan column tertentu

Jika kalian tidak menyebutkan columny otomatis mencari berdasarkan primary key, jika ingin lebih spesifik code seperti berikut.

<?php

$db->data_admin('username', 'admin');
Result: 

stdClass Object
(
	[id_admin] => 1
	[nama_admin] => admin
	[username] => admin
	[password] => 21232f297a57a5a743894a0e4a801fc3
)
